# Memorise

Memorise is a flutter app.
It send notification of your memories in daily basis

## Getting Started

### Project Structure
```
Lib
|
|---apis
|---repositories
|---models
|---blocs
|---pages
|---widgets
|---utils
```


### Install Dependency
---
```
flutter pub get
```

### Run Project
---
```
flutter run
```

### Build Apk
---
```
flutter build apk
```
